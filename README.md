<h1>Active Directory Home Lab</h1>

 ### [YouTube Demonstration](https://youtu.be/7eJexJVCqJo)

<h2>Description</h2>
In this project, I set up a virtual home lab environment using Oracle VirtualBox to simulate an Active Directory domain. This included installing and configuring a Windows Server to act as a Domain Controller, setting up DNS, and managing user authentication within the network. To automate administrative tasks, I used PowerShell scripting to create and manage 1,000 user accounts within Active Directory. This project demonstrates my skills in virtualization, network administration, and scripting automation.
<br />


<h2>Languages and Utilities Used</h2>

- <b>PowerShell</b> 
- <b>Diskpart</b>

<h2>Environments Used </h2>

- <b>Windows 10</b> (21H2)
